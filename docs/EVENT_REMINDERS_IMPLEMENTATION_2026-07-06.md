# Система напоминаний о событиях — 2026-07-06

## Поведение

- Событие через 8 и более дней: подтверждение сейчас, напоминание за 7 дней.
- Событие через 3–7 дней: подтверждение сейчас, напоминание при утреннем запуске следующего дня.
- Событие через 1–2 дня: только подтверждение и усиленный призыв создать открытку сейчас.
- Расчёт календарных дат выполняется в `Asia/Yekaterinburg`.

Подтверждение содержит детали и, когда запланировано второе письмо, ссылку отмены. Из-за автоматических сканеров ссылок отмена подтверждается отдельной кнопкой на странице.

## Надёжность

- Дубликаты формы объединяются по `dedupe_key`.
- Токен отмены стабильно выводится через HMAC; в базе хранится только SHA-256 хеш.
- Для HMAC обязателен отдельный production-секрет `REMINDER_TOKEN_SECRET`.
- Отправитель использует стабильные Resend idempotency keys для подтверждения и напоминания.
- Очередь атомарно захватывает записи через `FOR UPDATE SKIP LOCKED`.
- Зависшие записи `sending` возвращаются в обработку через 30 минут.
- После пяти неудачных попыток запись получает статус `failed`.
- Отменённые и `confirmation_only` записи в отправку не попадают.
