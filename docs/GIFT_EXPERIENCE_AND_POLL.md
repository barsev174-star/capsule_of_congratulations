# Финальный просмотр открытки и выбор подарка

## Открытие открытки

- По кнопке «Посмотреть что внутри» всегда запускается анимация конверта. Если пользователь уже открывал эту открытку, повторный просмотр можно принудительно запустить из режима предпросмотра.
- В анимации используется уменьшенная версия настоящей открытки, а не техническая заглушка. На сгибе конверта остаётся один знак Slovesto — конверт, выровненный по центру.
- Кнопка «Создать такую же открытку» на финальной открытке ведёт к управлению текущей открыткой (`/manage/...`), а не к форме создания новой.

## Вертикальное фото в шаблоне Paper Birthday

- Вертикальное фото размещается под PNG-ассетом полароидной рамки: изображение закрывает область внутри рамки без видимых границ по краям.
- На широком экране рамка использует всю доступную ширину правой колонки, чтобы не оставлять пустое место рядом с фото.
- Подпись находится в нижней бумажной части ассета и занимает не более двух строк. Длинный текст обрезается, не выходя за пределы рамки.

## Выбор подарка

Доступны два режима:

- **Варианты подарков** — участники выбирают один из карточек товаров.
- **Сбор бюджета** — участники видят варианты сумм для общего подарка.

При переходе в другой режим, если варианты уже заполнены, показывается подтверждение: «Сменить режим и начать заново? Текущие варианты будут заменены». Отмена сохраняет текущее состояние.

### Черновик и публикация

- Изменения вариантов сохраняются автоматически по актуальному состоянию формы; закрытие редактирования также запускает сохранение.
- Открыть голосование из черновика можно только после заполнения названия, вопроса и как минимум двух вариантов. Для ссылок на товары требуется `https://`.
- После запуска фиксируется дата закрытия голосования. На публичной странице участника отображается только опубликованное голосование.

## Миграция

Для хранения времени закрытия используется миграция `0019_gift_poll_closed_at.sql`. В окружениях с PostgreSQL её нужно применить стандартной командой:

```bash
npm run db:migrate
```

## Проверка перед выпуском

1. Открыть финальную ссылку с чистой сессией, нажать «Посмотреть что внутри» и убедиться, что сначала проигрывается анимация.
2. Проверить вертикальное фото с короткой и двухстрочной подписью на desktop и мобильном экране.
3. В обоих режимах выбора подарка добавить несколько вариантов, закрыть редактирование и обновить страницу.
4. Проверить подтверждение при смене режима и условия открытия голосования из черновика.
