## Критерий готовности этапа

Manager можно считать готовым для следующего production-deploy, когда:

1. на desktop удобно менять порядок блоков;
2. на mobile можно без боли заполнить основу, добавить поздравление, загрузить фото и назначить его в нужный блок;
3. финальная открытка показывает именно те фото и подписи, которые выбрал организатор;
4. build проходит;
5. production health check проходит после deploy.
