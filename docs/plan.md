# Главный план: максимум до деплоя, затем production и оплата

Дата актуализации: 2026-07-06.

## Цель

Сначала довести бесплатную версию локально настолько далеко, насколько это возможно без изменения production. Затем одним контролируемым релизом развернуть все накопленные изменения, полностью проверить production и только после стабилизации подключать оплату отдельным платёжным релизом.

Текущий статус этапа A: техническое ядро завершено 2026-07-06; остаются расширенный ручной аудит, production-зависимые email/AI проверки и решения владельца по тарифу, хранению, удалению и юридическим данным.

## Правило последовательности

1. До первого релиза не подключаем YooKassa и не принимаем деньги.
2. Первый релиз содержит всю локально подготовленную бесплатную beta.
3. После первого релиза проходим production-сценарии и устраняем только критические дефекты.
4. Оплата разрабатывается и включается отдельным релизом после подтверждения стабильности бесплатного сценария.
5. До платёжного релиза бесплатная публикация остаётся доступной. После включения оплаты новые открытки требуют 399 ₽ только для финальной публикации; существующие beta-открытки не должны случайно закрыться.
