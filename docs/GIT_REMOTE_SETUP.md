# Настройка GitHub remote

## Текущее состояние

Локальный git-репозиторий уже инициализирован.

На данный момент удаленный репозиторий GitHub еще не подключен, потому что сам проект на GitHub пока не создан.

## Что нужно от пользователя

1. Создать пустой репозиторий на GitHub.
2. Прислать URL репозитория.

Подойдут варианты вида:

1. `https://github.com/barsev174-star/<repo-name>.git`
2. `git@github.com:barsev174-star/<repo-name>.git`

## Что сделаю после этого

1. Подключу `origin`.
2. Проверю настройки ветки `main`.
3. Попробую выполнить `push`.
4. Если упрется в авторизацию, точно скажу, какой шаг нужен с твоей стороны.

## Рекомендуемый вариант

Если GitHub уже авторизован на этой машине, проще всего будет использовать HTTPS URL репозитория.
